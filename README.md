## Run Locally

Clone the project

```bash
    git clone https://github.com/JuneLin2001/chat-app.git
    cd chat-app
```
Setup .env file
```
    MONGODB_URI=...
    PORT=5001
    JWT_SECRET=...

    CLOUDINARY_CLOUD_NAME=...
    CLOUDINARY_API_KEY=...
    CLOUDINARY_API_SECRET=...

    NODE_ENV=development
```
Build and start the app
```
    npm run build
    npm start
```
